Calcul ta AAH

Ce projet a pour but:

- En premier lieu, de permettre a un foyer dont une personne qui perçois l'AAH, de connaître son plafond en fonction de l'évolution des revenus et de la composition du foyer.
- En second lieu, d'interagir avec d'autres personnes inscrite sur le site.

Pour la création du site "Calcul ta AAH"

- On l'a diviser en trois partie.

  • La première partie : Le Front-End (en cours de création)
  • La deuxième partie : Le Back-end ( donc ici)
  • la troisième partie : la base de donnée ( sur ElephantSQL)
- voici une ilustration simple :

Bienvenu dans le côté obscure, ici c'est le Back-end.

- Nous avons créé une API (le Back-end) pour interagir avec une base de données en SQL qui est en ligne sur ElephantSQL. en formule GRATUITE.
- Notre application Back-end est déployer sur Heroku en formule GRATUITE.

Les dépendances utilisées pour notre API

  • bcrypt
  • cors
  • dotenv
  • email-validator
  • express
  • jsonwebtoken
  • nodemon
  • pg
  • router
  • sanitizer
  • sequelize
  • yarn

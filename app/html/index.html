Calcul ta AAH

- Ce projet a pour but en premier lieu, de permettre a un foyer dont une personne perçois l'AAH, de connaître son plafond en fonction de l'évolution des revenus et de la composition du foyer.

- En second lieu, d'interagir avec d'autres personnes inscrite sur le site.

Bienvenu dans le côté obscure ici c'est le Back-end.

- Création d'une API pour interagir avec une base de données SQL qui est en ligne sur ElephantSQL.

Les dépendances utilisées pour notre API

  • bcrypt
  • cors
  • dotenv
  • email-validator
  • express
  • jsonwebtoken
  • nodemon
  • pg
  • router
  • sanitizer
  • sequelize
  • yarn

le Back-end

- Il est déployer sur Heroku

La base de donnée

- Elle est déployer sur le site [ElephantSQL](https://customer.elephantsql.com)
